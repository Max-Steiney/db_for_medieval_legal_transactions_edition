Bedingungen

Bedingungen

Eine Zeile pro gesuchter Person, Organisation oder Ort. Alle Slots sind optional; eine Zeile zaehlt als aktive Bedingung, sobald irgendein Slot gesetzt ist.

Alle aktiven Zeilen muessen gemeinsam erfuellt sein. Ohne mindestens eine aktive Zeile bleibt die Trefferliste leer.

Beruf, Tätigkeit oder Amt

Das Feld matcht den Beruf-, Tätigkeits- oder Amtstext, wie er ediert wurde (Substring, case-insensitiv). „Bürger" trifft auch „Bürgermeister".

Die Vorschlagsliste zeigt die 50 häufigsten Schreibungen. Das Eingabefeld selbst durchsucht aber alle im Korpus belegten Schreibungen (rund 1.500 distinkte Formen). Sobald der Suchbegriff auch außerhalb der Top-50 trifft, werden die zusätzlichen Treffer unten im Popover als „auch erkannt" angezeigt.
