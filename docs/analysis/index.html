Personen-Bedingungen

Personen-Bedingungen

Pro gesuchter Person eine Zeile in der Tabelle. Rolle ist Pflicht, Geschlecht und Beruf/Taetigkeit/Amt sind optional.

Alle Zeilen muessen gemeinsam erfuellt sein. Ohne mindestens eine Zeile mit gesetzter Rolle bleibt die Trefferliste leer.
