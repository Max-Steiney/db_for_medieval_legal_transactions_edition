Quellen

Anzahl der freigegebenen Quellen im Editionsstand. Eine Quelle ist eine einzelne TEI-XML-Datei mit einem Regest und seinen Annotationen.

Direkt aus den freigegebenen TEI-Quellen, gezählt pro Datei.

Nennungen

Wie oft Personen in den Quellen genannt werden. Eine Person, die in zwei verschiedenen Quellen vorkommt, zählt zweimal.

Aus den Personen-Annotationen in den freigegebenen TEI-Quellen. Mehrfachnennungen innerhalb derselben Quelle zählen einmal.

Events

Anzahl der annotierten Vorgänge in den Quellen. Ein Event ist ein konkreter Akt: Rechtsgeschäft, Siegelvermerk, Stadtbucheintrag oder Notiz.

Aus den Event-Annotationen in den freigegebenen TEI-Quellen, verschachtelte Events ausgeschlossen.
